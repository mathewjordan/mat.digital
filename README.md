# about

I am a designer and developer originally from Latah County, Idaho, United States. _Latah_ is a [Nez Percé](https://www.wikidata.org/wiki/Q1123923), (Nimíipuu) word for "the place of pine trees and pestle." I currently live in Asheville, North Carolina, and work for Northwestern University Libraries, where I contribute to open-source software enhancing digital archives with a focus on user experience.

Outside the office, I am likely to be found avoiding technology, tending a bonsai garden, backpacking the Blue Ridge Mountains, cooking, reading, and learning to become the best parent I can be.
